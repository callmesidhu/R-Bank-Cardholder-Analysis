# R-Bank-Cardholder-Customer-Segmentation

## Overview

This project performs Exploratory Data Analysis (EDA) and customer segmentation on bank cardholder data using R.

## Features

* Missing value analysis
* Statistical summary
* Outlier detection
* Correlation analysis
* Data visualization
* PAM clustering
* Customer segmentation

## Technologies Used

* R
* ggplot2
* tidyr
* cluster

## Results

The project identifies customer groups based on their banking behavior and provides cluster profiles for business insights.

## Dataset

Bank cardholder dataset (`pam_bank_cardholders.csv`)
